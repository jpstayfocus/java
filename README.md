# java-fundamental
fundamental of java: variables, loops, functions, and inheritance. 

# What is Java?
- Object oriented programming language
- created by James Gosling in 1995
- 3 billion devices run java programs
- similar to C++ but easier
- very popular

# What can you build with Java?
- Mobile apps for Android
- Desktop app using Java swing, JavaFX
- Enterprise applications
- Games such as [Minecraft](https://en.wikipedia.org/wiki/Minecraft)
- Cloud based applications
- and much more

### Compiler
- A compiler is a special program that processes statement written in a particular programming language and turns them into machine
  language or "Code" that a computer's processor uses.
